## 2. Color Palette

### Primary Colors

| Role | Name | Hex | Usage |
|---|---|---|---|
| **Primary Background** | Harris Navy | `#263E57` | Page backgrounds, headers, footers, navbars, dark panels |
| **Primary Accent** | Harris Red | `#B40000` | CTAs, highlights, accent bars, totals rows, badges |
| **Primary Text (on dark)** | White | `#FFFFFF` | All text on navy or red backgrounds |

### Secondary / Supporting Colors

| Role | Name | Hex | Notes |
|---|---|---|---|
| **Logo Gold** | Equipment Yellow | `#E8A020` | Pulled directly from the logo's wheel loader — use sparingly for icon accents or callout highlights |
| **Logo Cream** | Parchment | `#F0E6CC` | Logo badge background — use as a warm off-white for document paper backgrounds or card fills |
| **Body Text (on light)** | Near-Black | `#1A1A1A` | Body copy on white/light backgrounds |
| **Light Background** | Off-White | `#F7F7F7` | Table row alternates, content area backgrounds |
| **Border / Divider** | Light Gray | `#D9D9D9` | Table borders, section dividers on light backgrounds |

### Color Rules
- **Never** place Harris Red (`#B40000`) text on Harris Navy (`#263E57`) — low contrast, fails accessibility.
- White on Navy: ✅ | White on Red: ✅ | Near-Black on Off-White: ✅
- The Equipment Yellow (`#E8A020`) is a logo color only — do not use it as a primary UI color or text color. Use it for icons, decorative lines, or star ratings.
- Avoid using green as a highlight color in client-facing documents (the green total cell in older estimates is **not** a brand color and should be replaced with Harris Red + White text going forward).

## 3. Typography

### Primary Typeface
| Role | Font | Style | Notes |
|---|---|---|---|
| **Display / Headers** | **Norwester** | Regular (it is a single-weight display font) | Used for all major headings, the company name in the logo, and prominent UI labels. Free download: fontsquirrel.com/fonts/norwester |

### Body / Supporting Typefaces
Since Norwester is display-only, pair it with a clean, geometric sans-serif for body and UI text:

| Role | Font | Weight | Notes |
|---|---|---|---|
| **Body Text** | **Inter** | 400 Regular, 600 SemiBold, 700 Bold | Free on Google Fonts. Geometric, modern, highly legible. Matches the brand's tech-forward personality. |
| **Fallback** | System-UI / -apple-system / Segoe UI | — | Use in web apps when custom fonts aren't loaded |

### Type Scale (documents and apps)

| Level | Font | Size | Weight | Color |

| Section Heading (H2) | Norwester | 18–22pt | Regular | `#263E57` or `#FFFFFF` |
| Sub-heading (H3) | Inter | 13–15pt | Bold (700) | `#263E57` or `#FFFFFF` |

## 4. Logo

### Description
The Harris Excavation logo is a **retro hexagonal badge** featuring:
- A detailed yellow/gold wheel loader (front-end loader) illustration facing left
- A cream/parchment inner background
- A dark navy outer border (`#263E57`)
- A red mid-band (`#B40000`) carrying the **HARRIS** wordmark in white
- A navy lower band carrying **EXCAVATION** in spaced white caps and ***California*** in white script
- **"2024"** and forward-pointing chevron arrows (`>>>>>>>`) in the upper right of the badge, in navy
- Overall aesthetic: vintage americana / retro industrial badge

### Logo File
- **Primary file:** `Harris_Excavation_Logo_1_.png`
- Provided as a PNG on white/transparent background
- Always use the provided file — do not attempt to recreate the logo

### Logo Usage Rules
- **Minimum size:** No smaller than 80px wide in digital contexts; 1 inch wide in print
- **Clear space:** Maintain a margin equal to the height of the "H" in HARRIS on all sides
- **Placement in documents:** Center-aligned at the top of the page header, flanked by horizontal navy rule lines (see reference document layout)
- **On dark backgrounds:** The logo has a cream badge background — it works on both light and dark surfaces
- **Do not:** Recolor, stretch, add drop shadows, place on busy photographic backgrounds, or modify any element of the badge

### Logo in Document Header (standard layout)
```

```
Full-width navy rules extend left and right of the centered logo badge.

## 5. Document Standards

### Reference Document: Budget Proposal (Morgan Hill)
The Morgan Hill Site Development Budget is the canonical example of Harris Excavation's client-facing document style. All proposals and reports should match this template.

### Page Layout
- **Paper:** US Letter (8.5" × 11")
- **Margins:** 1" all sides
- **Header:** Logo centered with navy horizontal rules flanking (left and right of logo)
- **Footer:** Two-zone bar — Left zone `#263E57` with social proof ("25+ 5 Star Reviews" + Google logo), Right zone `#B40000` with "CSLB # 1117960" in white Norwester/bold text

### Document Sections (Proposal/Bid)
1. **To / Date block** — Left-aligned recipient info, right-aligned date in bold caps (e.g., APRIL 7, 2026)
2. **Salutation** — First names, bold
3. **Intro paragraph** — Warm, direct, personalized. Bold weight. 3–5 sentences.
4. **Pricing Table** — Full-width, navy header row, clean alternating rows, Harris Red accent on Total row (white bold text on red)
5. **Scope of Work sections** — Underlined bold heading per scope item, bullet list of specific work included
6. **Qualifications** — Bulleted, bold
7. **Exclusions** — Bulleted, bold
8. **Closing paragraph** — Brief, warm sign-off (e.g., "We look forward to helping you build your dreams!")
9. **Signature Block** — Left: handwritten signature image + "David Harris / Owner" + credentials. Right: email / phone / website with icons.

### Pricing Table Format
| Column | Alignment |
|---|---|
| Item Description | Left |
| Unit Price | Right |
| Quantity | Right |
| Unit | Center |
| Total | Right |

- Header row: `#263E57` background, `#FFFFFF` text, Inter Bold
- Data rows: alternating `#FFFFFF` / `#F7F7F7`
- Total row: `#B40000` background, `#FFFFFF` bold text
- Borders: `#D9D9D9` light gray, 1px

## 6. Brand Voice & Tone

### Core Personality Traits
- **Confident** — We know what we're doing. No hedging, no fluff.
- **Direct** — Get to the point. Say exactly what's included, what's excluded, and what it costs.
- **Personable** — We're a small owner-operated business. Use first names. Be human.
- **Tech-forward** — Reference GPS grade control, digital project management, and modern tools when relevant. We are the "new generation of Dirt World contractors."
- **Gritty / Earned** — Credibility comes from getting on job sites and doing the work. Let the work speak.

### Tone by Context

| Context | Tone | Example |
|---|---|---|
| Client proposals | Warm + professional | "It was awesome getting to meet you and walk the property..." |
| Website copy | Confident + educational | "We leverage the latest technology to ensure your project is completed on time." |
| Blog (Full Buckets Only) | Conversational + expert | Insider knowledge, no jargon walls, written like a knowledgeable friend |
| Internal tools (Harris Estimator) | Functional + clear | Labels, tooltips, and error messages should be plain and direct — no corporate-speak |
| Social media | Casual + proud | Show the work. Short captions. Let the machines and the results do the talking. |

### Voice Rules
- ✅ Use first person ("we," "our team," "I") — this is a personal brand
- ✅ Use the client's first names in proposals
- ✅ Use plain construction language — no overly academic or legalese phrasing
- ✅ Include social proof naturally ("25+ 5-Star Reviews") without being pushy
- ❌ Never use vague contractor clichés ("best in class," "second to none," "one-stop shop")
- ❌ Never overpromise on timelines or costs — qualify with "budget," "estimate," "allowance" as appropriate
- ❌ Never use all-lowercase in professional documents — maintain proper capitalization

### Tagline Usage
- **Full:** "Build Smarter. Dig Deeper."
- **Split use:** "Build Smarter." or "Dig Deeper." as standalone emphasis lines (section headers, social posts)
- **Never** paraphrase or modify the tagline

## 7. UI & App Design Standards

*Apply these when building the Harris Estimator, internal dashboards, web apps, or any digital tool.*

### Color Tokens (CSS variables)
```css
:root {
  --color-bg-primary:     #263E57;  /* Navy — main backgrounds, sidebars, navbars */
  --color-bg-surface:     #F7F7F7;  /* Off-white — content areas, cards */
  --color-bg-paper:       #FFFFFF;  /* White — inputs, table cells */
  --color-accent:         #B40000;  /* Red — CTAs, active states, badges, alerts */
  --color-accent-gold:    #E8A020;  /* Gold — logo accent, star ratings, icon highlights */
  --color-text-on-dark:   #FFFFFF;  /* White text on navy/red */
  --color-text-primary:   #1A1A1A;  /* Near-black body text on light */
  --color-text-muted:     #6B7A8D;  /* Muted text, labels, placeholders */
  --color-border:         #D9D9D9;  /* Dividers, table borders */
  --color-border-strong:  #263E57;  /* Navy borders for emphasis */
}
```

### Component Conventions
- **Buttons (primary):** `#B40000` background, `#FFFFFF` text, Norwester or Inter Bold, 4px border-radius
- **Buttons (secondary):** `#263E57` background, `#FFFFFF` text, same radius
- **Buttons (ghost):** `#FFFFFF` background, `#263E57` border + text
- **Active / selected state:** `#B40000` left border or background highlight
- **Input fields:** White background, `#D9D9D9` border, `#263E57` focus ring
- **Table headers:** `#263E57` background, `#FFFFFF` text
- **Total / summary rows:** `#B40000` background, `#FFFFFF` bold text
- **Badges / status chips:** Norwester, `#B40000` or `#263E57` fill, white text

### App Header Pattern (Harris Estimator)
```
[#263E57 full-width bar]
  Left: Harris Excavation Logo (small)
  Center or Left: App name in Norwester white
  Right: User info / settings icon
```
